```
SELECT
    IFNULL(
        (
            SELECT DISTINCT(salary)
            FROM employee
            ORDER BY salary DESC
            LIMIT 1, 1
        ), NULL) as SecondHighestSalary
```



### ISNULL(x, 0)
- x 값이 NULL이면 0을, NULL이 아니면 x 값을 출력
